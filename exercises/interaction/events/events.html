Keyboard & Mouse Event Demo

Keyboard Events

I have added a keyboard event handler to this webpage. Here's the code:


function handleKey(event) {
    const code = event.code;
    const key = event.key;
    console.log(`code: ${code} and key: ${key}`);
}

document.addEventListener("keydown", handleKey);

This script prints useful keyboard event values to the JavaScript console. Open your browser's developer console and try pressing some keys.

Note: The browser window needs to be in focus. If your keystrokes are going into the console instead, click anywhere on the page first.

Try letter keys, arrow keys, numbers, punctuation, and more.

Mouse Click Events

Suppose you want to process mouse clicks. The event you want to bind is, unsurprisingly, click. Here’s the code: 


function onMouseClick(event) {
    const mx = event.clientX;
    const my = event.clientY;
    console.log(`click at (${mx}, ${my})`);
}

document.addEventListener("click", onMouseClick);

This code logs the mouse click position in the browser window.

Open the JavaScript console, make sure the document is in focus, and click around the page. Try clicking near the corners:

  • Top-left → small x and y values
  • Top-right → large x, small y
  • Bottom-left → small x, large y
  • Bottom-right → large x and y
